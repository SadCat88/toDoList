## Описание

Перечень используемых технологических решений в проекте:

- Сборка проекта через Webpack 4
- Для сборки JSDoc документации используется Gulp
- HTML через Pug
- CSS через SCSS
- Vanilla JavaScript через Babel с трансплитом до ES5
- Git (спасибо, Кэп)
